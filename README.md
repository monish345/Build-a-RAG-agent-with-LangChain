# 📄 LangChain RAG System

This project implements a Retrieval-Augmented Generation (RAG) system using LangChain.

## 🚀 Features
- PDF document loading
- Text chunking
- Vector embeddings using HuggingFace
- FAISS vector database
- Question answering using LLM

## ⚙️ How it works
PDF → chunks → embeddings → FAISS → retrieval → LLM → answer

## 🧠 Tech Stack
- Python
- LangChain
- FAISS
- HuggingFace Transformers

## ▶️ Usage
Run notebook in Google Colab or Python environment and upload a PDF.
